버튼 형식 태그

input type="submit"

        
        -form 태그에 작성한것 action 에 지정한 주소로 제출하는 버튼
        -텍스트와 아이콘을 함께 넣기 어렵고, 꾸미기 제한적

button 관련 태그

        기본타입이 submit 으로 form 태그에 작성한것 action 에 지정한 주소로 제출하는 버튼
         type 을 submit 으로 안해도 제출하는게 기본설정
        -종류: submit 제출, button 클릭후 제출이나 초기화 이외 기능 설정 , reset 초기화
        -이미지나 아이콘 넣어 버튼 꾸밀수 있다
