입력관련태그 활용편

select - option 태그

            select 태그
            사용자가 선택하도록 드롭다운목록 생성
            option 태그
            무조건 select 태그 안에 존재
            개발자가 정의된 옵션을 지정해주는 태그

예제코드

field set - legend 태그

        field set: 여러개의 입력태그 그룹화
        (ex)form 태그 안에서 관련항목끼리 그룹화
        legend: 무조건 field set 태그 안에 존재, field set 태그의 제목표시
                field set 태그에 포함된 태그그룹 설명
