block 형식

      화면공간 수직분할, 태그 닫으면 줄바뀜
      width, height 와 같은크기 지정 속성 사용가능

      block 형식의 태그(태그 닫히면 자동줄바뀜)
      h1~h6 제목, p/pre 문단, hr 수평선, div 영역분할, 
      시맨틱 (div 태그에 이름을 넣는태그)

inline 형식태그

      화면 수평분할
      가로에서 사용할 만큼만 공간차지
      width, height 와 같은 크기지정속성 불가

      inline 형식의 태그
      b 굵게, strong 굵게, i 기울이기, mark 형광팬, span 영역분할

크기지정 가능여부

  • block: 크기지정 가능
  • inline: 크기지정 불가
        -inline 특징 수평분할
        -block 특징 크기지정 가능
        -대표적 태그: img, iframe 
        *iframe 태그
        -웹문서(html) 내부에 또다른 웹문서 추가
