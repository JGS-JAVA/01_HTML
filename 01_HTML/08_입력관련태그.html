form 태그 / infut 태그

form 태그

        입력양식 작성공간
        내부 입력값을 지정된 곳으로 제출
        [속성]
        1 action 제출될 곳 지정
        2 method 설정된 곳으로 데이터 전달
        3 name 이름지정 (javascript 와 함께 사용)
        4 target 제출된 페이지를 현재/새 탭으로 열기

input 태그

        웹문서에서 사용자(클라이언트)가 입력할수 있는 양식제공
        많은 type(속성)이 있고, input 의 변형형태도 많다
        *input 태그는 name 필수작성!
         name 으로 이름끼리 서로 연결할때 필요

textarea 태그

      여러줄의 텍스트 입력가능
      [속성]
      rows: 기본 세로줄 수 지정
      cols: 기본 가로너비 수 지정
      placeholder: 힌트
      maxlength: 최대 문자수
      readonly: 읽기전용
      disabled: 비활성화된 상태
