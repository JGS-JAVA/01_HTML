화면배치(표시,형식) : display 속성

        요소가 화면에 어떻게 보여질지 형식지정
        -block : 화면을 수직분할 (행나눔, 한줄다차지)
            width, height 지정가능
        -inline : 화면 수평분할 (하나의 행의 열을 나눔 == 글자, 문자열 생각)
        -inline-block : inline 의 수평분할 + block 의 width, height 지정
        -noe : 요소는 존재하나 화면에는 보이지 않게함
        -flex : 요소의 정렬되는 방향, 요소 간의 간격을 유연하게 처리
        -grid : 격자무늬 형식으로 화면 쉽게분할

block 형식의 요소를 inline 형식으로 변경
