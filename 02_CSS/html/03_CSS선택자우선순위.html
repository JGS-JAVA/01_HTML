CSS 선택자 우선순위

      css속성은 style 태그 또는 css 파일에 작성된 순서대로 해석됨
      같은 요소에 여러 css 속성 설정시 우선순위대로 적용
      1. 동일한 우선순위면 제일 마지막 것이 반영
      2. 여러 선택자를 이용해서 같은 요소 선택해도 css 속성 다르면 모두반영
      3. 태그와 스타일이 가까울수록 우선적용
      4. 스타일.css 파일과   태그 안에 작성한 css 는 동일한 우선순위나 
      가장 마지막것이 반영
      (예) 
      
      마지막 것인 style 태그가 반영됨.
      
      (예) 
        
        마지막 것인 link 태그가 반영됨.
        1순위 : css속성 : 속성값!important;-애초에 변경불가로 잠금
        2순위 : inline-style 속성-태그 옆에 직접 스타일 작성해서
        
인라인 스타일 속성

        3순위 : #아이디 속성명 -특정 아이디에만 스타일 줄때
        4순위 : .클래스 속성명 -주로 반복적 디자인 적용
        5순위 : 태그이름
