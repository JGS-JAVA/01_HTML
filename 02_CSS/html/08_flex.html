Flex 레이아웃

        display : flex;
        div 안에 작성한 태그들을 유연하게 배치할 수 있는 속성
        수평이나 수직정렬 가능
        div (블록형태, 가로전체차지) 를 inline (부분차지) 형태로 바꾸는 효과있음
        함께 쓰이는 속성
        /가로로 존재하는 태그의 여백설정/
        1. justify-content:center; 플렉스 컨테이너 내 자식태그를 가로방향으로 가운데정렬
        2. justify-content:space-around; 플렉스 컨테이너 내의 자식태그들 사이 균등하게 여백주기
        
        /세로로 존재하는 태그 줄의 여백 가운데로 설정/
        1. align-items : center; 세로줄에서 가운데 정렬. 
        주로 flex-direction:row; 와 함께 사용할 때 세로 가운데 정렬 필요할 경우 사용 
        text-align-items
        └ 가로 ┴ 세로 ┘ 

        /한 줄에서 있을 수 있는 태그의 너비가 넘어가면 자동으롤 줄바꿈 되도록 설정/
        1. flex-wrap : wrap;
        최대한 가로에서 세로로 줄바꿈이 되어서 태그가 넘어가지 않도록 설정너비가 넘으면
        어쩔수 없이 줄바꿈 됨.

flex와 inline 차이

특징 요소 배치 방식 크기 제어 응용 범위 정렬 옵션
inline 같은줄에 나열되며 필요한 공간만 차지 높이,너비 지정 어려움 정렬 옵션 제한적 텍스트 및 링크 요소 등에 사용
flex 가로,세로방향으로 유연하게 배치 크기, 여백 쉽게 제어 가능 다양한 정렬옵션 지원 전체 레이아웃 설계 주로 사용
