display 속성

        diplay 속성 html 요소 배치방식을 설정하는 중요한 css 속성
        요소: 태그 + 내용
        dilplay 주요 값으로 
        [block, inline, inline-block, flex, grid, inline-flex]
        모든 태그에는 display 가 각 속성에 맞게 기본설정이 되어있다.
        diplay : block = div, h1~6, p 등 대표적 block설정으로 존재
        (예) h1 태그 만들 때 diplay:block 주고 font-weight:bold 주고
        font-size:32px 하고 color:black 기본설정으로 만들자!
        위 h1 태그처럼 모든 태그에 css 기본설정되어있음
        display:inline = span, a, strong 등이 대표적으로 inline 설정으로 존재
        (예) span 태그 만들 때 : display:inline 주고 font-size:16px color:black 
        기본설정으로 함.

        각 태그에 기본스타일이 적용되어 있는데 기본스타일의 틀을 변경하길 원할 경우
        css 를 이용해서 디자인 변화를 준다.
   
        display : inline-block = inline 처럼 한줄에 표시됨
        image 와 button 태그가 inline-block 으로 설정됨

display:flex

flex 요소는 내부 항목들이 수평 또는 수직으로 유연하게 정렬
        inline 과 flex 차이
        inline= 가로 세로 조정 X
        flex = 가로 세로 조정 O

display:grid

행과 열로 내부항목 정렬
        주로 이미지나 웹툰처럼 틀과 형식으로 규칙적인 정렬이 필요할 때
        많이 사용
        grid-template-columns: 1fr 1fr; 
         fr:fraction(부분, 일부)
         그리드 내에서 사용가능한 공간을 비율로 나누는 단위
         1fr: 그리드의 사용가능공간을 같은 비율로 나눠서 차지
         1fr 1fr : 두 개 열이 동일하게 공간을 나눠서 차지
         1fr 2fr : 첫번째 열은 가로전체 공간의 1/3 차지
                    두번째 열은 가로전체의 2/3 차지
        
                    gap: 10px; 그리드 셀 사이에 10px 간격추가. 
        그리드 안에서 자동으로 10px 간격주고 자동배치함
