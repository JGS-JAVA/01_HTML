태그 선택자

        현재 문서의 특정태그 모두찾아 선택
        [작성법]
        
            태그명 {
                css 코드;
            }

id(#) 태그 선택자

        HTML 문서 내에서 id 
        일치하는 요소 선택
        [작성법]
        
            #id이름 {
                css 코드;
            }

class(.) 선택자

        html 문서내에서 class 속성값 일치하는 
        모든요소 선택
        style에선 class 주로 사용
        id값 주로 자바스크립트
        class를 디자인에서 많이 사용해야함
        [작성법]
        
            .class이름명 {
                class 코드;
            }
