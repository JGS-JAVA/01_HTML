<2>일반 구조(위치)선택자 
        형제관계 있는 요소중 특정요소만 선택하는 선택자
        위치기준으로 구문
        A:first-child -> 형제중 첫째가 A 면 선택 
        A:last-child -> 형제중 마지막이 A 면 선택 
        A:nth-child(n=숫자) -> 형제중 n번째면 A 면 선택 
        A:nth-last-child -> 형제중 뒤에서부터 n번째면 A 면 선택
