배열 Array

        변수를 묶음으로 다룬다(자료구조 기초)
        js변수특징: 대입되는 값에 따라 자료형 변함
        js배열: 자료형 무관히 여러값 한번에 다룸
        자료형 제한X
        길이 제한X

        js배열 선언방법
        const arr = new Array(); //0칸짜리 배열(비어있는 배열) 생성
        const arr = new Array(3); //3칸짜리 배열(비어있는 배열) 생성
        const arr = []; //0칸짜리 배열(비어있는 배열) 생성 []는 배열기호
        const arr = [값1, 값2, 값3]; //3칸짜리 배열(채워진 배열) 생성 []는 배열기호
        index : 각 자리 위치
        length : 총 길이
