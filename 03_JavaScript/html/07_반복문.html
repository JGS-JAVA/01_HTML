for 문

        코드반복
        일정횟수만 반복한다.(무한X)

        for([1]초기식 ; [2]조건식; [4]증감식){
            [3] 조건식이 true 이면 반복수행할 코드;
        }
        1~4 까지 1회반복 후
        2~4 계속 반복
        조건식 false 면 바로 종료
        초기식: for 문 제어하는 용도의 변수 선언
        조건식: for 문 반복여부 지정
               true 는 반복 false 는 종료
        증감식: 초기식에 사용된 값 변환
                변환된 초기식에 의해 조건식 결과 달라짐

이중 for 문 (중첩반복문)

      반복문 내 반복문 작성
      메모리 사용량 많아서 사용지양하기
      
      for(초기식1; 조건식1; 증감식1){
        for(초기식2; 조건식2; 증감식2)
      }

while 문

      끝이 정확히 정해지지 않은 반복에서 주로 사용
      (예)어떤 값이 입력되기 전까지 반복
      while(조건식){
        조건식이 true 이면 반복할 코드;
      }
      조건식 true 이면 내부코드 1회 반복후 다시 조건식 검사
      false 나올때까지 무한반복
