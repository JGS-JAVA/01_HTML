        window.prompt("글자")

href 와 src 차이점
        둘 다 링크, 리소스 참조시 사용하나 그 용도가 다름
        href : 페이지 이동, 기능 참조 / 주로 css파일, 페이지 간 이동, 외부 웹 페이지 링크 등
        src : 페이지 안에 외부 소스 삽입 / 주로 이미지, 동영상, js 파일 등
