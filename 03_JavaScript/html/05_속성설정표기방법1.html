        DOM(Document Object Model)

        특정 태그 내에 작성된 아이디값이나 클래스값으로 접근할 수 있다. 
        
        document 하위요소
        id - value, style, innerhtml
        classname - length, item, nameitem
        tagname - length, item, nameitem
