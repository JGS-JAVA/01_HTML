for 문

        코드반복
        일정횟수만 반복한다.(무한X)

        for([1]초기식 ; [2]조건식; [4]증감식){
            [3] 조건식이 true 이면 반복수행할 코드;
        }
        1~4 까지 1회반복 후
        2~4 계속 반복
        조건식 false 면 바로 종료
        초기식: for 문 제어하는 용도의 변수 선언
        조건식: for 문 반복여부 지정
               true 는 반복 false 는 종료
        증감식: 초기식에 사용된 값 변환
                변환된 초기식에 의해 조건식 결과 달라짐
