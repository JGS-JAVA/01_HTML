인터프리터와 컴파일러

        프로그래밍 언어로 작성한 코드를 컴퓨터가 이해할 수 있는 형태로 변환하는 프로그램
        compiler : 코드 수집. 사람의 언어로 코딩한 전체소스코드를 한번에 컴퓨터의 언어로 변환하고
                  변환이 완료된 파일을 생성하여 파일 실행해서 빠르다.   
                  사람이 만든 파일.java -> 사람이 만든 파일.class 파일이 실행되어 빠르다
                  예시 언어:c, c++, c#, java
        interpreter : 해석하다. 소스코드를 한줄씩 해석하면서 실행해서 느리다
                    코드 작성후 바로실행 가능
                    한줄씩 해석하며 실행해서 컴파일러보다 느리다
                    예시 언어: python, javascript, ruby
