Script 언어란?

        기본 프로그램 동작을 사용자요구에 맞게 수행해준다
        자바는 별도 소스코드 컴파일(이진수번역) 한다. 그래서 빠르다.
        자바스크립트는 Interpreter(코드해석기계) 이용해 코드 바로실행한다
        컴파일 소요시간 없음, 프로그램 실행시간 상대적으로 느림

        프로그래밍 언어로 작성한 코드를 컴퓨터가 이해할 수 있는 형태로 변환하는 프로그램

JavaScript 란?

      웹 브라우저(크롬, 엣지, 웨일, 파이어폭스 등)에서
      주로 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어

      ECMA Script 표준을 따르는 웹기술
      (JS 버전: ES5, ES6, ES7, ...)

JavaScript 코드 작성법

  1. 브라우저 개발자도구 -> console 에 직접작성
  2. HTML 내부에 script 태그 이용해 작성
    body head 등 html 태그 내부에서
    <script> 태그를 이용해서 작성한다는 의미
  3. HTML 외부에 존재하는 .js 파일을 연결해서 사용하는 방법
    .css 와 비슷하게 사용
    .css 와 달리 <script src="파일명.js"> 파일을 불러옴
  4. 태그에 직접 JS코드 작성방식 (비추천)
    CSS 스타일 직접 꾸미기방식
