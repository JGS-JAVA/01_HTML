조건문 (Condition)

      Condition = 어떤 조건(상태)에 따라 다르게 동작
      조건 true 면 작업수행
      코드 해석순서 제어하는 제어문의 종류로
      조건식에 따라 수행되는 코드변경구문

if(만약에)

      
        if (조건식) {
          조건식이 true 이면 실행할 코드;

        }
      
      *조건식: true, false 인 식(연산)
      *조건문: 조건결과에 따라 실행을 제어하는 문장 (if, switch)

입력받은 수가 양수인지 아닌지 검사

if(만약에) - else (아니면) 문

      
        if(조건식) {
          조건식이 true 이면 실행할 코드;
        }
        else {
          조건식이 false 이면 실행할 코드;
        }
      

      if-else : 수행할 구문 많을때
      삼항연산자: 수행할 구문 적을때
      else 에 조건식 작성X

if(만약에) - else if(아니면) - else(아니면)

      
        if (a) {
          true 면 실행;
        }
        else if (b) {
          a 가 false, b 가 true 면 실행;
        }
        else {
          a, b 모두 false 면 실행;
        }
        else if 많이 써도 됨

switch 문

switch, case, default

      switch 문
      식 하나의 결과로 많은 경우의 수 처리할 때 사용하는 조건문
switch(식) { } default 위 모든 케이스에 대해 false 일때 수행 (if 와 else) = (case 와 default)

조건이 여러개인 switch case

  if문처럼 조건이 여러개면
  -> if(조건1 || 조건2 || 조건3) {  코드   } 와 비슷하다
  case 조건1: case 조건2: case 조건3: 
                                      
      조건1이나 조건2나 조건3일 때 부합하면 실행할 코드
      break;
  case 조건4: case 조건5:
  조건4나 조건5일때 부합하면 실행할 코드변경구문
  break;
  default:
  위 모든 경우에 false 이면 실행할 코드
  break;
