복합 대입 연산자 (+=,-=,*=,/=,%=)

        기존 변수에 저장된 값에 새로 대입되는 값의 산술연산결과 대입

        증감 연산자는 1씩 증감, 복합대입 연산자는 원하는대로 증감

        let temp = 10;//초기값 설정
        temp += 5;    // temp = temp + 5
        결과 : temp = 15

현재값 : 0

+= (더하기 연산자)

      기존 값에 새값 더하거나 이어붙여 업데이트 한다
      == (동등 연산자) 자동타입수정
      += (더하기 연산자) 글자끼리 이어붙이고, 숫자끼리 (연산으로)더해준다
      let A = "감"
      let B = "자"
      A += B; -> 감자;
      let C = 5
      let D = 5
      C += D; -> 10;
      let E = "5";
      let F = 5;
      E += F -> 10 또는 55 나올수 있음
