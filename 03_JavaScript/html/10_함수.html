매개변수 Parameter, 전달인자 Argument

        매개변수 : 
        parameter 변경될 기준을 제공하다
        para 근처, 옆에 제공하다 
        meter 기준 
        중간역할 하거나 전달하는 변수
        함수 선언시 괄호 안에 작성
        전달인자로부터 받은 값 저장
        함수 정의부분에서 사용가능

        
        /중매 매
        /개입 개
        /변할 변
        /셀 수

        전달인자 : 
        제공되는 정보, 실제 값
        함수가 호출될 때 argument 값들이 증거처럼 파라미터에 전달됨
        /전할 전 :정보 전하다
        /전달될 달 :전달 완료
        /끌어당길 인 :특정위치로 이끌다
        /아들 자 :특정 목적이나 역할 가진 요소

        요소 : 태그 + 태그안 내용
