정규표현식 RegularExpression, RegExp

        정규식 = 정규표현식
        특정 규칙 가진 문자열 집합 표현하는 형식언어
        입력된 문자열에 대한 특정조건 검색, 일치여부 판단, 치환에 대한 조건문을
        간단히 처리한다

정규식 표현

        문자열 패턴 나타냄
        문자마다 뜻이 있다
        일반 문자열 : 같은 문자열 존재여부 검색
        /abc/ : abc 라는 문자열 존재여부 검색

        ---------------------------------------
[ ] : 대괄호 안 어떤 문자와 일치
/[abc]/ : a,b.c 중 하나라도 일치하는 문자유무 검색
        [0-9] : 0~9 까지 모든 수 사용가능
        [a-z] : 모든 영어 소문자
        [A-Z] : 모든 영어 대문자
        [ㄱ-힇] : ㄱ~힇 까지 모든 한글
        [가-힇] : 가~힇 까지 모든 한글 (자음, 모음 제외)
        -----------------------------------------------
        [^] : 대괄호 안의 문자와 불일치
        [^abcd] : a,b,c,d 와 모두 불일치
        ^ (캐럿) : 문자열의 시작
        /^a/ : a 로 시작
        $ (달러) : 문자열의 끝
        /z$/ : z 로 끝
        ---------------------------------------
        escape 문자
        특정 문자를 특별 용도로 사용

        \ + 특수문자 : 기능이나 패턴이 아닌 일반문자 그대로 인식
        \^ : ^ 특수문자 존재유무 확인

        \d (digit, 숫자) : 아무 숫자 (0~9 중 하나)
        \w (word, 문자) : 아무 문자 (띄어쓰기, 특수문자, 한글 등 다른문자 불가)
        \s (space, 공간) : 아무 공백 문자 (띄어쓰기, 엔터, 택 등 가능)
        ---------------------------------------
        * 수량관련문자
        a{5} : a 문자 5개 존재여부
        a{2,5} : a 2개 이상 5개 이하
        a{2,} : a 개 이상

정규표현식 표현

      1. 정규식 객체 생성법
      1) const regExp = new RegExp;
      2) const regExp = /정규표현식/;

      2. 문자열 패턴 확인 기능(메서드=함수)
      1)regExp.test(문자열)
      문자열에 정규식과 일치하는 패턴 있으면 true, 없으면 false
      2)regExp.exec(문자열)
      문자열에 정규식과 일치하는 패턴 있으면 처음 매칭되는 문자열 반환, 없으면 null 값 반환
